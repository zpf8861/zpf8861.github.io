利用github+HEXO搭建自己的博客

By 赵鹏飞

发表于 2016-02-27

文章目录
  1. 1. 步骤概述

有很多种方法可以写博客，一般最开始使用的是类似CSDN的博客网站，然后你会希望拥有一个自己的服务器去维护博客，这种方法不错，但是需要付出一定的金钱，并且需要承担一部分运维工作。
当然，像本文这样，将写博客当作提交代码一样，既方便使用又容易维护，只需要前期简单的配置博客的环境。
github是什么，不需要过多介绍，[HEXO]是什么，大家可以去点击连接去看下官网。
下面介绍如何快速的在github上搭建HEXO博客框架。

步骤概述

按照下面顺序会很清晰快捷的完成搭建工作

  • 在github上设置ssh密匙

  • 在github上建立一个空的仓库，并且clone到本地一个文件夹

  • 尝试进行提交readme.md文件，如果成功证明ssh密匙设置正确

  • 安装HEXO环境，初始化一个工程

  • 修改根目录下的配置文件_config.yml 尤其是

  • 之后分别使用

将工程部署到github
